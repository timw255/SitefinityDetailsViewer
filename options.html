Sitefinity CMS Detail Viewer

Options

Highlight Content Blocks on Front-End:

Enabling this feature will highlight content areas on the front-end of the website. Useful for diagnosing instances where the content inside a content block is bigger than the allowed size.


Label Content Placeholders:

Enabling this feature will turn on content placeholder labels in the admin area of the site. There's a button, to the left of the label text, that allows you to collapse/expand a container to access the area behind it.


Placeholder Options

Initial Visibility:

Regardless of this setting, pressing "z" will hide/show the labels


Overlay Color:

Set a custom color for the collapse animation and collapsed state


Preview Pages:

Enabling this feature will turn on page previews in the admin area of the site. Hover over the title of published pages to view.
